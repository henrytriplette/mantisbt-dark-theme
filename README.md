# MantisBT Dark Theme

A darker theme for Mantis bug tracking software: http://mantisbt.org/

## Installation

1. Copy the css folder into Mantis root folder.
2. Add

```
$g_css_include_file = "/css/custom/default.css";
```
to the config_inc.php file.
3. ???
4. Profit!
